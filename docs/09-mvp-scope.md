# docs/09-mvp-scope.md

## MVP target: 2–4 weeks of focused build
### Must-have
- Travel between nodes on a 2D map
- Vehicle with 6–8 subsystems (simplified)
- Crafting + repair
- 3 biome types + hazards
- 1–2 puzzle templates
- 1 “mystery beat” chain (mini arc)
- Seeded deterministic generation
- Vehicle sprite changes per module (can be simple overlays)

### Nice-to-have
- Factions and trade
- Notebook UI
- Share seed/build codes
- More puzzle templates

### Avoid early
- Combat
- Large crew simulation
- Too many resources
- Too many UI screens

## MVP success criteria
Player finishes a “mini deepening” arc and says:
- “Oh. This is going somewhere bigger.”
- “My vehicle looks and plays differently than my last run.”
- “I want to share this seed/build.”
