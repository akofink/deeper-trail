# docs/07-content-packs.md

## Content pack idea
Allow adding new:
- Biomes
- Hazards
- Items
- Modules
- Puzzle templates
- Encounter tables
- Story beats

## Format
JSON or TS modules with validation.
Keep deterministic generation stable by versioning pack IDs.

## Goal
Community modding without servers:
- Players share packs + seeds
- Everything still runs locally
